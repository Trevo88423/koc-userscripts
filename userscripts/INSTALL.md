## PC / Mac (Desktop Browsers)

### Chrome / Edge / Brave

1. **Install Tampermonkey extension**
   - Go to [Chrome Web Store - Tampermonkey](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)
   - Click "Add to Chrome"

2. **Install the script**
   - Click this link: [Install KoC Data Centre](https://raw.githubusercontent.com/Trevo88423/koc-userscripts/main/userscripts/KoC-DataCentre.user.js)
   - Tampermonkey will open and show the script
   - Click "Install"

3. **Done!** Visit [Kings of Chaos](https://www.kingsofchaos.com) and log in

### Firefox

1. **Install Tampermonkey extension**
   - Go to [Firefox Add-ons - Tampermonkey](https://addons.mozilla.org/en-US/firefox/addon/tampermonkey/)
   - Click "Add to Firefox"

2. **Install the script**
   - Click this link: [Install KoC Data Centre](https://raw.githubusercontent.com/Trevo88423/koc-userscripts/main/userscripts/KoC-DataCentre.user.js)
   - Tampermonkey will open and show the script
   - Click "Install"

### Safari (Mac)

1. **Install Userscripts extension**
   - Go to [App Store - Userscripts](https://apps.apple.com/us/app/userscripts/id1463298887)
   - Install the app (free)
   - Open Safari > Preferences > Extensions > Enable Userscripts

2. **Install the script**
   - Click the Userscripts icon in Safari toolbar
   - Click "Open Directory" to find your scripts folder
   - Download the script from GitHub and save it to that folder
   - Or use the "New Script" option and paste the code

## iOS (iPhone / iPad)

### Option 1: Userscripts App + Safari (Recommended)

1. **Install Userscripts app**
   - Download from [App Store - Userscripts](https://apps.apple.com/us/app/userscripts/id1463298887) (free)

2. **Enable the extension**
   - Open Settings > Safari > Extensions
   - Enable "Userscripts"
   - Set permissions to "Allow" for kingsofchaos.com

3. **Set up script directory**
   - Open the Userscripts app
   - Tap "Set Userscripts Directory"
   - Choose a folder in iCloud Drive or On My iPhone (e.g., create a "Userscripts" folder)

4. **Install the script**
   - Open Safari and go to: `https://raw.githubusercontent.com/Trevo88423/koc-userscripts/main/userscripts/KoC-DataCentre.user.js`
   - Tap the Share button > "Save to Files"
   - Save to your Userscripts directory
   - Rename the file to `KoC-DataCentre.user.js` if needed

5. **Enable the script**
   - In Safari, tap the puzzle piece icon (extensions)
   - Tap Userscripts
   - You should see the script listed - make sure it's enabled

### Option 2: Orion Browser

1. **Install Orion Browser**
   - Download from [App Store - Orion](https://apps.apple.com/us/app/orion-browser-by-kagi/id1484498200)
   - Orion supports Firefox/Chrome extensions on iOS

2. **Install Tampermonkey**
   - In Orion, go to Settings > Extensions
   - Install Tampermonkey from the extension gallery

3. **Install the script**
   - Navigate to the script URL
   - Tampermonkey will handle installation

## Updating the Script

### Automatic Updates
Tampermonkey checks for updates automatically. To force an update:
1. Click the Tampermonkey icon
2. Go to Dashboard
3. Click the script name
4. Click "Check for updates" or the refresh icon

### Manual Update
1. Go to the script URL
2. Tampermonkey will detect the update
3. Click "Update" or "Reinstall"

## Troubleshooting

### Script not running?
- Make sure you're logged into KoC
- Check Tampermonkey is enabled (icon should be colored, not greyed out)
- Check the script is enabled in Tampermonkey dashboard
- Try refreshing the page

### Features missing?
- Some features require visiting specific pages first (e.g., Armory to calibrate multipliers)
- Check browser console (F12) for any error messages

### Mobile issues?
- Make sure you're using a supported browser (Firefox or Kiwi on Android, Safari with Userscripts on iOS)
- The default Chrome/Safari apps don't support userscripts without extensions
